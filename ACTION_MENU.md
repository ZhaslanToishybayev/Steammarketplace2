# 🚀 Выберите действие:

## 📋 Действия:

### 1️⃣ **Steam OAuth в браузере**
```
В браузере открыть:
• http://localhost:5173
• Нажать "Войти через Steam"
• Проверить авторизацию
```

### 2️⃣ **Настроить домен**
```
• Узнать ваш IP (curl ifconfig.me)
• В панели ps.kz: Добавить A-записи
• Привязать домен к домашнему IP
• Проверить: https://yourdomain.ps.kz/api/health
```

### 3️⃣ **Port Forwarding**
```
• В настройках роутера
• Пробросить порты 80/443
• Полный внешний доступ
```

### 4️⃣ **Посмотреть логи Steam бота**
```
• Проверить статус бота
• Увидеть инвентарь
• Проверить активность
```

### 5️⃣ **Подготовить миграцию на VPS**
```
• Создать .env.production
• Подготовить Docker файлы
• Инструкции по заказу VPS
• Автоматический деплой скрипт
```
