## 🎯 Ваши данные:
- **IP адрес:** 45.140.25.16
- **Домен:** [ВАШ_ДОМЕН] (скажите какой)

---

## 📋 Пошаговый гайд:

### ШАГ 1: Скажите ваш домен
Напишите: "Мой домен: example.com"

### ШАГ 2: В панели ps.kz
1. Зайти на https://ps.kz
2. Войти в аккаунт
3. "Домены" → "Мои домены"
4. Выбрать ваш домен
5. "DNS записи" → "Добавить запись"

### ШАГ 3: Добавить A-записи
Добавить ДВЕ записи:

**Запись 1:**
```
Тип: A
Имя: @
Значение: 45.140.25.16
TTL: 3600
```

**Запись 2:**
```
Тип: A  
Имя: www
Значение: 45.140.25.16
TTL: 3600
```

### ШАГ 4: Сохранить и подождать
- Нажать "Сохранить"
- Подождать 5-15 минут

### ШАГ 5: Проверить
```bash
# В браузере:
https://yourdomain.ps.kz/api/health
https://yourdomain.ps.kz/api-docs
```

## ✅ Готово! Домен привязан

После этого:
- Ваш сайт будет доступен по домену
- Steam OAuth будет работать (с localhost ключом)
- Можно тестировать полную функциональность

## 🔧 Если хотите SSL (HTTPS):

### Вариант 1: Let's Encrypt (бесплатно)
После привязки домена:
```bash
# На сервере с доменом
sudo apt install certbot python3-certbot-nginx
sudo certbot --nginx -d yourdomain.com -d www.yourdomain.com
```

### Вариант 2: Cloudflare (проще)
1. Перенести DNS на Cloudflare (бесплатно)
2. Включить SSL в настройках
3. Автоматический HTTPS
