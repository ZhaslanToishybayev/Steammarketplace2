# 📊 СЛЕДЕНИЕ ЗА ЛОГАМИ

## 🔍 ПОСМОТРЕТЬ ЛОГИ СЕЙЧАС:
```bash
tail -f steam-monitor.log
```

## ⏰ СЛЕДУЮЩИЕ ПРОВЕРКИ:
- **Каждые 10 минут** скрипт будет проверять Steam
- **Логи обновляются** автоматически
- **При успехе** появится:

```
✅ STEAM BOT ГОТОВ К РАБОТЕ!
✅ Инвентарь: 45 предметов
🚀 Автозапуск приложения...
```

## 📱 КОМАНДЫ:
```bash
# Следить в реальном времени
tail -f steam-monitor.log

# Посмотреть последние 20 строк
tail -20 steam-monitor.log

# Проверить, работает ли
ps aux | grep "steam-bot-monitor" | grep -v grep

# Остановить
pkill -f "steam-bot-monitor"
```

## ⏱️ КОГДА ЗАРАБОТАЕТ:
- **Rate limit снимается:** 1-3 часа
- **Мониторинг проверит:** каждые 10 минут
- **Будет логировать:** каждую попытку
