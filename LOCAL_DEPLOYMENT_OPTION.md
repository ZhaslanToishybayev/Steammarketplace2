# 💻 Развертывание на домашнем компьютере

## ✅ МОЖНО ЛИ? ДА! Но есть нюансы...

### 🎯 Преимущества:
- ✅ Экономия денег (не нужно платить за VPS)
- ✅ Полный контроль
- ✅ Можете тестировать прямо сейчас
- ✅ Домен можно привязать

### ⚠️ Ограничения:

#### 1. **Статический IP (самая большая проблема)**
```bash
# Узнать свой IP
curl ifconfig.me
# Если меняется - НЕ ПОДХОДИТ для продакшна
```

#### 2. **Порт форвардинг на роутере**
Нужно открыть порты 80 и 443:
- Зайти в настройки роутера
- Настроить перенаправление портов
- 80 → ваш_IP:3001
- 443 → ваш_IP:3001

#### 3. **24/7 работа**
- Ноутбук должен быть включен всегда
- Расход электроэнергии
- Риск перегрева

#### 4. **Безопасность**
- Открытые порты в домашней сети
- Меньше защиты чем на VPS

#### 5. **Скорость интернета**
- Нужна хорошая исходящая скорость
- Рекомендуется: 50+ Mbps

## 🚀 Как развернуть НА НОУТБУКЕ:

### Шаг 1: Подготовка
```bash
# 1. Клонировать проект
git clone <your-repo> steam-marketplace
cd steam-marketplace

# 2. Скопировать production файлы
cp .env.example .env.production
```

### Шаг 2: Настройка .env.production
```env
NODE_ENV=production
PORT=3001
BASE_URL=https://yourdomain.ps.kz
CLIENT_URL=https://yourdomain.ps.kz

# Домашний IP (узнать через ifconfig.me)
BASE_URL=https://ВАШ_IP_ДОМАШНИЙ:3001
CLIENT_URL=https://ВАШ_IP_ДОМАШНИЙ:3001

MONGODB_URI=mongodb://localhost:27017/steam-marketplace
REDIS_URL=redis://localhost:6379

# Генерировать секреты
JWT_SECRET=ваш_секрет_64_символа
SESSION_SECRET=ваш_секрет_32_символа

STEAM_API_KEY=ваш_ключ
```

### Шаг 3: Запуск (БЕЗ Docker)
```bash
# Установить зависимости
npm install

# Запустить MongoDB (если есть)
# или использовать MongoDB Atlas
# или запустить локально

# Запустить Redis (если есть)
# или использовать внешний Redis

# Запустить приложение
npm run build
NODE_ENV=production node app.js
```

### Шаг 4: Настройка домена
В панели ps.kz добавить A-запись:
```
Тип: A
Имя: @
Значение: ВАШ_ДОМАШНИЙ_IP
TTL: 3600
```
