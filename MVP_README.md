## 🧪 **ТЕСТИРОВАНИЕ MVP:**

### **1. Проверить здоровье сервера:**
```bash
curl http://localhost:3001/health
```
**Ожидаемый ответ:**
```json
{
  "status": "OK",
  "timestamp": "2025-11-02T19:02:22.644Z"
}
```

### **2. Получить список скинов:**
```bash
curl http://localhost:3001/api/mvp/listings
```

### **3. Добавить скин на продажу:**
```bash
curl -X POST http://localhost:3001/api/mvp/listings \
  -H "Content-Type: application/json" \
  -d '{
    "itemName": "AK-47 | Redline",
    "weaponType": "AK-47",
    "price": 45.99,
    "sellerId": "demo_user"
  }'
```

### **4. Купить скин:**
```bash
curl -X POST http://localhost:3001/api/mvp/purchase/listing_id \
  -H "Content-Type: application/json" \
  -d '{
    "buyerId": "buyer_demo",
    "buyerName": "Demo Buyer"
  }'
```

## 🎮 **ФУНКЦИИ MVP:**

### ✅ **Реализованные:**
- 🌐 **Веб-интерфейс** - Bootstrap + современный дизайн
- 🛒 **Торговля** - Покупка/продажа в демо режиме

## 🆘 **ТРОУБЛШУТИНГ:**

### **Сервер не запускается:**
```bash
# Остановить все процессы Node
killall -9 node

# Запустить заново
npm start
```

### **MongoDB не подключается:**
```bash
# Запустить MongoDB
sudo systemctl start mongod
# или
mongod --dbpath /data/db
```

### **Порт занят:**
```bash
# Найти процесс на порту 3001
lsof -i :3001

# Завершить процесс
kill -9 <PID>
```
