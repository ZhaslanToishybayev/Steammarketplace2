# 🔐 Объяснение проблемы со Steam OAuth

## Раньше (когда работало):
```
┌─────────────────────────────────────────────────────────┐
│ 1. Приложение работало на: https://sgomarket.com        │
│ 2. Steam API Key регистрировался для: sgomarket.com     │
│ 3. redirect_uri: https://sgomarket.com/api/auth/...     │
│ 4. Steam: "Всё ок, домены совпадают" → ✅ РАБОТАЕТ      │
└─────────────────────────────────────────────────────────┘
```

## Сейчас (когда НЕ работает):
```
┌─────────────────────────────────────────────────────────┐
│ 1. Приложение работает на: http://localhost:3001        │
│ 2. Steam API Key зарегистрирован для: sgomarket.com     │
│ 3. redirect_uri: http://localhost:3001/api/auth/...     │
│ 4. Steam: "Ошибка! Домены НЕ совпадаются" → ❌ НЕ РАБОТАЕТ│
└─────────────────────────────────────────────────────────┘
```

## Что произошло:
1. Вы переключились с production (sgomarket.com) на development (localhost)
2. Steam API Key остался тот же (зарегистрирован для sgomarket.com)
3. Steam строго проверяет соответствие доменов
4. localhost ≠ sgomarket.com → "Invalid URL"
