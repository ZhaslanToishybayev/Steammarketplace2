## 🚀 ЗАПУСК (30 сек)

### Вариант 1: Обычный запуск
```bash
node steam-bot-monitor.js
```

### Вариант 2: В фоне (РЕКОМЕНДУЕТСЯ)
```bash
nohup node steam-bot-monitor.js > monitor.log 2>&1 &
```

### Вариант 3: Со screen/tmux
```bash
screen -S steam-monitor
node steam-bot-monitor.js
# Нажать Ctrl+A, затем D для отсоединения
```

---

## 📊 ЧТО ДЕЛАЕТ СКРИПТ

### Проверяет каждые 10 минут:
1. ✅ **MongoDB** - подключение к базе
2. ✅ **Steam бота** - подключение и инвентарь
3. 📝 **Логирует** все в `steam-monitor.log`
4. 🚀 **Автозапуск** основного приложения при успехе

### При успехе:
- Показывает: `✅ STEAM BOT ГОТОВ К РАБОТЕ!`
- Автоматически запускает: `npm start`
- Завершает мониторинг

## 📋 КОМАНДЫ ДЛЯ РАБОТЫ

### Посмотреть логи в реальном времени:
```bash
tail -f steam-monitor.log
```

### Посмотреть последние 50 строк:
```bash
tail -50 steam-monitor.log
```

### Остановить мониторинг:
```bash
pkill -f "steam-bot-monitor"
```

### Проверить, работает ли:
```bash
ps aux | grep "steam-bot-monitor" | grep -v grep
```

## ⚙️ НАСТРОЙКИ

В файле `steam-bot-monitor.js` можно изменить:

```javascript
const CONFIG = {
  CHECK_INTERVAL: 10 * 60 * 1000,  // 10 минут (можно 5 или 15)
  LOG_FILE: 'steam-monitor.log',   // Файл логов
  MAX_RETRIES: 100,                // Максимум попыток
  START_APP_ON_SUCCESS: true       // Автозапуск приложения
};
```
