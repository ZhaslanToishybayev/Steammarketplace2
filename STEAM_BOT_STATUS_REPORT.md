## 🗄️ ИНВЕНТАРЬ

### ❌ Обнаруженные проблемы:

**КРИТИЧЕСКАЯ ОШИБКА:**
```
❌ Ошибка: client.getInventory is not a function
```

**Причина:** Метод `getInventory()` недоступен в используемой версии SteamUser (4.29.3). В современных версиях библиотеки инвентарь получается через TradeOfferManager.

**Текущий код (services/steamBot.js строки 85-86):**
```javascript
getInventory(client) {
  client.getInventory(730, 2, (err, inventory) => {
```

**Правильный подход:**
```javascript
// Инвентарь загружается автоматически через TradeOfferManager
this.manager.on('inventoryLoaded', () => {
  const items = this.manager.inventory.getItems();
});
```
