### 🧪 **Как тестировать Steam OAuth:**

1. **Проверка системы:**
   - Перейдите на: http://localhost:3000/auth/steam
   - Нажмите "Проверить систему" для диагностики

2. **Тест аутентификации:**
   - На странице http://localhost:3000/auth/steam нажмите "Пробный вход"
   - Или на главной auth странице нажмите "Войти через Steam"

3. **Проверка API Key:**
   - Используйте кнопку "Проверить API Key" на Steam OAuth мониторинге
