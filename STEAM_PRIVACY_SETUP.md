# 🔓 Как сделать инвентарь Steam публичным

## 📋 Пошаговая инструкция:

### 1. Войдите в Steam
- Откройте Steam Client или веб-версию
- Войдите в свой аккаунт

### 2. Перейдите в Настройки приватности
- В правом верхнем углу нажмите на ваш аватар
- Выберите **"Account Details"** или **"Настройки аккаунта"**
- Найдите раздел **"Privacy Settings"** или **"Настройки приватности"**

### 3. Настройте приватность инвентаря
Найдите следующие настройки и установите **"Public"** (Публичный):
- ✅ **Inventory:** Public
- ✅ **Game Details:** Public
- ✅ **Friends List:** Public (опционально)

### 4. Сохраните изменения
- Нажмите **"Save Changes"** или **"Сохранить"**
- Подтвердите действие если требуется

### 5. Очистите кеш Steam (если нужно)
В Steam Client:
- **Steam** → **Settings** → **Web Browser** → **Delete Web Browser Cache**

---

## 🎮 Альтернатива: Использовать только Bot Inventory

Если не хотите делать инвентарь публичным, система покажет:
- ✅ **Bot Inventory** - 1 предмет (AUG | Sweeper)
- ❌ **Ваш инвентарь** - недоступен (приватный)

---

## 🔧 Техническое объяснение:

**Проблема:** Steam OpenID 2.0 не предоставляет access token для Steam Community API.
Без токена API не может получить приватный инвентарь пользователя.

**Решение:** Либо публичный инвентарь, либо ограниченный функционал только с ботом.

---

*Инвентарь должен быть публичным для работы с Steam API без токена доступа.*
