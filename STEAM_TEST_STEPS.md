## 🔑 ШАГ 1: Тестирование Steam OAuth

### В браузере:
1. Открыть: **http://localhost:5173**
2. Найти кнопку "Войти через Steam"
3. Кликнуть и авторизоваться

### Или напрямую:
1. Перейти: **http://localhost:3001/api/auth/steam**
2. Должен редиректить на Steam

## 🌐 ШАГ 3: Настройка внешнего доступа (домен)

### Вариант A: Привязка к домашнему IP

1. **Узнать свой IP:**
```bash
curl ifconfig.me
```

2. **В панели ps.kz:**
   - Зайти в "Домены" → "Мои домены"
   - Выбрать ваш домен
   - "DNS записи" → "Добавить запись"
   ```
   Тип: A
   Имя: @
   Значение: ВАШ_IP
   TTL: 3600
   
   Тип: A
   Имя: www
   Значение: ВАШ_IP
   TTL: 3600
   ```

3. **Подождать 5-10 минут** и проверить:
```bash
https://yourdomain.ps.kz/api/health
```

### Вариант B: Port Forwarding (для полного доступа)

1. **В настройках роутера:**
   - Войти в админ-панель (обычно http://192.168.1.1)
   - Найти "Port Forwarding" или "Перенаправление портов"
   - Добавить правила:
     ```
     Внешний порт: 80 → Внутренний: 3001
     Внешний порт: 443 → Внутренний: 3001
     Протокол: TCP
     ```

2. **Проверить доступность:**
```bash
curl http://ВАШ_IP/api/health
```
