## 🎯 Что можно протестировать:

### ✅ Steam OAuth Аутентификация
1. Нажмите "🔑 Steam OAuth Аутентификация"
2. Вас перенаправит на Steam сайт
3. Подтвердите вход
4. Вернетесь в интерфейс с токеном

### ✅ Просмотр инвентаря
1. Сначала авторизуйтесь через Steam OAuth
2. Выберите игру (CS:GO, Rust, Dota 2, TF2)
3. Нажмите "📦 Проверить Инвентарь"
4. Система загрузит ваш реальный Steam инвентарь

### ✅ Торговля
1. Скопируйте свой Steam Trade URL
2. Вставьте в поле "🔗 Steam Trade URL"
3. Выберите игру
4. Нажмите "🔄 Создать тестовое предложение"
5. Бот Sgovt1 обработает вашу заявку

### ✅ API Эндпоинты
1. Тестируйте все API кнопками слева
2. Проверяйте ответы и статусы
3. Убеждайтесь в работоспособности
