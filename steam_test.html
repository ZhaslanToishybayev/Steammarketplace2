Для полноценного тестирования Steam интеграции требуется:

📋 Доступные Endpoint'ы

🔐 Аутентификация

GET /api/auth/steam - Вход через Steam

GET /api/auth/steam/return - Callback

POST /api/auth/refresh - Обновить токен

🏪 Инвентарь

GET /api/inventory - Список предметов

POST /api/inventory/sync - Синхронизация

GET /api/inventory/stats - Статистика

🎮 Торговля

GET /api/trading/offers - Список предложений

POST /api/trading/offers - Создать предложение

GET /api/trading/bots - Статус ботов

🚀 Как тестировать Steam интеграцию:

  1. Получите Steam API ключ: steamcommunity.com/dev/apikey
  2. Обновите .env файл: Замените STEAM_API_KEY=dev-steam-api-key на настоящий ключ
  3. Перезапустите backend: cd apps/backend && npm run start:dev
  4. Пройдите OAuth: Нажмите "Войти через Steam"
  5. Тестируйте API: Используйте кнопки выше для проверки функционала
