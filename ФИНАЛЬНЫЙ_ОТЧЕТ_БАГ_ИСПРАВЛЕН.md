## 🔧 ЧТО БЫЛО ИСПРАВЛЕНО

### 1. **Архитектурная проблема** (services/steamBot.js)
**Проблема:** Бот загружал ТОЛЬКО CS2 инвентарь

**Решение:** Переписана структура инвентаря и метод loadInventory()

```javascript
// НОВАЯ СТРУКТУРА
this.inventory = {}; // { '730': [], '570': [] }
this.inventoryLoaded = {}; // { '730': false, '570': false }

// НОВЫЙ МЕТОД
async loadInventory(appId = null) {
  if (!appId) {
    // Загружаем ОБА инвентаря
    this.inventory['730'] = await this.loadInventory(730); // CS2
    this.inventory['570'] = await this.loadInventory(570); // Dota 2
  } else {
    // Загружаем конкретную игру
    this.inventory[appId] = await steamIntegration.getBotInventory(this.manager, appId);
  }
}
```

### 2. **Фильтрация по играм** (routes/steam.js:375-380)
**Проблема:** Dota 2 фильтр проверял только `item.tradable`

**Решение:** Добавлена проверка `item.appid === 570`

```javascript
// ИСПРАВЛЕННЫЙ КОД
} else if (gameName === 'Dota 2') {
  // Dota 2 - show ONLY Dota 2 items (appId=570)
  filteredItems = botInventory.filter(item => {
    return item.appid === 570 && item.tradable;  // ✅ Теперь правильно!
  });
}
```
